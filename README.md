# Dimitrios Devliotis Portfolio

Static GitHub Pages portfolio for [azzurre.github.io](https://azzurre.github.io/).

## What is included

- Modern responsive portfolio for employers
- Light and dark theme toggle with saved preference
- Curated project cards backed by live GitHub repository data
- Employer-focused case studies for the strongest AI, computer vision, and ML projects
- Direct CV link at `/assets/resume.pdf`
- Contact links for email, LinkedIn, and GitHub

The site is intentionally static, so it can be served directly from GitHub Pages without a build step.
